📖 QUICK START (5 MINUTES)

1. Install Dependencies:
   npm install

2. Setup Database:
   createdb taskforge
   node scripts/migrate.js

3. Configure .env.local:
   DATABASE_URL=postgresql://...
   NEXTAUTH_SECRET=...
   GITHUB_CLIENT_ID=...
   GITHUB_CLIENT_SECRET=...

4. Run:
   npm run dev

5. Open:
   http://localhost:3000

Full details in QUICKSTART.md
