## 🎬 Production Checklist

### Pre-Recording
- [ ] Clean browser cache
- [ ] Close unnecessary browser tabs
- [ ] Hide bookmarks bar
- [ ] Set zoom to 100%
- [ ] Prepare demo data (projects, issues)
- [ ] Create dummy files for upload demo
- [ ] Test all features work
- [ ] Practice the flow 2-3 times

### Recording Setup
- [ ] Set screen resolution to 1920x1080
- [ ] Turn off notifications
- [ ] Close Slack, Discord, email
- [ ] Ensure stable internet
- [ ] Test microphone levels
- [ ] Start recording software
- [ ] Record in a quiet room

### During Recording
- [ ] Speak clearly and slowly
- [ ] Pause between actions
- [ ] Show cursor movements smoothly
- [ ] Don't rush
- [ ] If you make a mistake, pause and restart that section

### Post-Production
- [ ] Cut mistakes and long pauses
- [ ] Add background music
- [ ] Add text overlays for features
- [ ] Add annotations (circles, arrows)
- [ ] Add transitions between scenes
- [ ] Color correct if needed
- [ ] Normalize audio levels
- [ ] Add intro/outro cards
- [ ] Export at 1920x1080, 30fps

### Final Checks
- [ ] Watch full video
- [ ] Check audio sync
- [ ] Verify all features shown
- [ ] Check video quality
- [ ] Check audio quality
- [ ] Get feedback from team
- [ ] Make final adjustments

## 💡 Pro Tips

1. **Keep cursor movements smooth** - Don't jerk the mouse around
2. **Pause after each action** - Give viewers time to absorb
3. **Use keyboard shortcuts visibly** - Show what keys you're pressing
4. **Smile while narrating** - It comes through in your voice
5. **Record in segments** - Easier to fix mistakes
6. **B-roll footage** - Record extra clips of features for editing
7. **Test on different devices** - Ensure readability
